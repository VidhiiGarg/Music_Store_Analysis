##  Overview

This project analyzes a music store database using SQL to generate business insights related to customers, sales, artists, and genres.

## Tools Used
SQL
PostgreSQL
PgAdmin4

## Key Insights
Top-selling music genres
Best customers by revenue
Highest revenue-generating countries
Most popular artists and albums

## SQL Concepts Used
Joins
Aggregate Functions
Subqueries
CTEs
Window Functions

##  Project Goal

The goal of this project is to strengthen SQL skills by solving real-world business problems using data analysis.
